项目背景

化工控制实训装置的设计初衷与教学目标

TCT_100 化工控制实训装置是嵌入式软件开发实践课程的综合实训项目。项目以化工安全监控为应用场景，模拟化工厂中常见的环境参数监测与设备控制需求。

装置基于 ESP32-C3（RISC-V 架构）微控制器，集成多种传感器、执行器与通信模块，采用 FreeRTOS 多线程架构实现任务分离与优先级管理，通过 MQTT 协议实现物联网远程监控能力。

通过本项目，学生可以系统性地掌握：嵌入式 GPIO 编程、传感器驱动开发、实时操作系统 (RTOS) 多线程编程、WiFi 网络通信、MQTT 物联网协议、JSON 数据序列化、以及工业级控制逻辑设计。

ESP32-C3 FreeRTOS Arduino Framework MQTT WiFi 嵌入式开发 物联网

FreeRTOS 多线程架构

三线程分离设计，保证关键任务实时响应

优先级: 高 (1) → 低 (0)

主线程 (优先级1)

按键 · 传感器 · 继电器 · OLED · 串口

│
线程一 (优先级0)

WiFi维护 + MQTT维护 + 消息发布

线程二 (优先级0)

多彩LED指示灯状态机

🔒 线程同步: portMUX_TYPE 互斥量保护 MQTT publish 操作
📡 主线程写入共享状态 → LED线程只读 (ESP32 原子操作，无需锁)

技术栈与依赖

项目使用的核心框架与第三方库

🔧

Arduino Framework

基于 ESP32 Arduino Core 3.x，提供标准 Arduino API 与 ESP-IDF 底层能力。

⏱️

FreeRTOS

ESP32-C3 原生集成。使用 xTaskCreate 创建独立线程，vTaskDelay 非阻塞延时，portMUX 互斥量同步。

📡

PubSubClient

轻量级 MQTT 客户端库，支持 QoS 0，15 秒 KeepAlive，JSON 消息体 (ArduinoJson)。

🌈

Adafruit NeoPixel

WS2812 单总线 RGB LED 驱动，24 位色深，支持 HSV 色彩空间与亮度调节。

🖥️

U8g2 (SH1106)

OLED 显示驱动库，I²C 接口，支持中文 GB2312 字库，128×64 分辨率。

🌡️

DallasTemperature

DS18B20 一线数字温度传感器驱动，支持异步温度转换，精度 ±0.5°C。
