User Guide

使用指南

从开机到远程控制，完整的 TCT_100 操作手册

快速上手

5 分钟让 TCT_100 运转起来

第1步：上电启动

通过 USB-C 接口供电（5V/1A），装置自动启动。RGB 灯执行开机动画（七彩循环 + 呼吸灯），OLED 显示 "Starting..."。

第2步：WiFi 配网

首次开机无保存凭据时自动进入配网模式。串口监视器（115200 baud）中列出扫描到的 WiFi 列表，输入序号选择网络，输入密码完成连接。凭据自动保存至 Flash，下次开机自动重连。

第3步：MQTT 连接

WiFi 连接成功后，自动连接预设的 MQTT Broker (47.98.170.180:8081)。连接成功后订阅命令主题并上报初始状态。可通过串口命令 mqtt show 查看配置、mqtt set 修改参数。

第4步：合上总闸

按下 KEY1（总闸开关），系统进入运行状态。默认自动模式，传感器开始采集数据，装置根据光照和温度自动控制灯光与风扇。

第5步：观察运行

OLED 屏幕实时显示各项参数。RGB 灯根据系统状态自动切换颜色模式。可通过串口查看定期上报的传感器数据，或通过 MQTT 客户端远程监控。

按键操作说明

两个物理按键实现全部本地控制

🔑

KEY1 — 总闸开关

短按：切换总闸开/关状态。

总闸合上时强制进入自动模式，所有继电器复位为 00 状态。

总闸断开时所有继电器关闭，RGB 灯显示红灯常亮。

🎛️

KEY2 — 模式切换

长按 2 秒：切换自动/手动模式。

短按（手动模式）：循环切换继电器组合状态：

00 (全关) → 10 (灯亮) → 01 (风扇) → 11 (全开)

⚠️ 自动模式下短按无效

串口命令参考

通过 USB 串口 (115200 baud) 发送以下命令

命令功能示例
forget清除保存的 WiFi 凭据并断开forget
reboot重启设备（1秒后）reboot
rainbowRGB 彩虹效果 5 秒（调试用）rainbow
mqtt show打印当前 MQTT 配置mqtt show
mqtt set broker <ip>修改 MQTT Broker 地址mqtt set broker 192.168.1.100
mqtt set port <num>修改 MQTT 端口mqtt set port 1883
mqtt set user <name>修改 MQTT 用户名mqtt set user admin
mqtt set pass <pwd>修改 MQTT 密码mqtt set pass 123456
mqtt set devid <id>修改设备 IDmqtt set devid PCT100001

MQTT 远程控制

通过 MQTT 客户端向设备下发 JSON 指令

🔌

控制继电器

Topic: chemctrl/{deviceId}/command

{"cmd":"set_relay","relay":3,"value":true}

relay=3 灯光, relay=4 风扇

🔄

切换模式

{"cmd":"set_mode","mode":"auto"}
{"cmd":"set_mode","mode":"manual"}

📊

查询状态 & 设阈值

{"cmd":"get_status"}
{"cmd":"set_threshold","temp":32.0,"light":200}

自动模式 vs 手动模式

特性自动模式手动模式
继电器控制传感器数据自动决策按键/MQTT 手动切换
KEY2 短按无效（提示信息）循环切换 4 种组合
传感器阈值生效（滞回控制）不生效
MQTT 控制仅 set_mode/get_status完整控制权限
RGB 灯根据传感器告警变色继电器状态决定颜色
