产品定位

化工装置智能控制器是一款基于 ESP32-C3 微控制器的嵌入式教学平台。它模拟化工场景中常见的安全监控需求，集成多种传感器与执行器，支持本地自动控制与远程 MQTT 通信。

学生通过本装置可掌握：

✅ 光照/温度传感器数据采集

✅ 继电器逻辑控制（灯光、风扇）

✅ RGB 多彩指示灯状态显示

✅ WiFi 联网与 MQTT 物联网通信

✅ FreeRTOS 多线程编程

✅ OLED 显示屏驱动与 UI 设计

ESP32-C3 FreeRTOS MQTT WiFi

六大核心功能

覆盖传感器采集、自动控制、状态指示与远程通信的完整实训链路

🌡️

温度监控

DS18B20 数字温度传感器，精度 ±0.5°C。支持动态设置上下限阈值，越界自动触发风扇控制，阈值持久化存储于 Flash。

💡

光照检测

高精度光敏电阻 (LDR)，0~1000 lux 量程。自动判定光照强度，低于阈值自动开灯，高于阈值自动关灯，支持滞回控制。

🔌

继电器控制

双路继电器分别控制灯光与风扇。支持自动模式（传感器决策）与手动模式（按键/MQTT指令切换），4 种组合状态 (00/01/10/11)。

🌈

多彩LED指示

Adafruit NeoPixel RGB 灯，6 种显示模式直观反映设备状态：严重警报、光照越界、温度越界、MQTT断开、WiFi断开、正常运行。

📡

WiFi + MQTT

2.4GHz WiFi 联网，支持 Web 配网与凭据记忆。MQTT 协议远程上报状态（温度、光照、继电器、模式），接收云端控制指令。

🖥️

OLED 显示屏

128×64 SH1106 OLED 屏幕，中文显示。实时展示 WiFi IP、运行模式、传感器数值、阈值设定、灯光/风扇状态。

多彩LED 警报模式

RGB 指示灯根据系统状态自动切换，直观反馈设备运行情况
