Technical Specifications

技术规格

化工装置智能控制器完整硬件参数、传感器指标与通信协议

硬件规格

装置核心硬件组件及参数一览

组件 型号 / 规格 说明
主控芯片 ESP32-C3 RISC-V 32位单核, 160MHz, 400KB SRAM, 4MB Flash, 集成 WiFi 4 + BLE 5.0
温度传感器 DS18B20 数字一线接口, 量程 -55°C ~ +125°C, 精度 ±0.5°C, 分辨率 9~12 位可调
光照传感器 光敏电阻 LDR 模拟输入, 12位ADC采样 (0~4095), 换算范围 0~1000 lux
RGB LED Adafruit NeoPixel (WS2812) 单总线控制, 24位色深 (RGB各8位), 亮度可调 0~255
OLED 显示屏 SH1106 128×64 I²C 接口 (SDA=4, SCL=5), 单色白光, 支持中文显示 (GB2312字库)
继电器 ×4 5V 电磁继电器 GPIO2/3/6/7 驱动, 分别控制 relay1/relay2/灯光/风扇
按键 ×2 KEY1 (GPIO20), KEY2 (GPIO21) KEY1: 总闸开关 (轮询检测), KEY2: 模式切换 (外部中断 + 长短按识别)
板载 LED GPIO9 状态指示: 灯光或风扇任一开启时点亮

GPIO 引脚分配

ESP32-C3 引脚功能定义

GPIO 功能 方向 备注
GPIO 0RGB LED (NeoPixel)OUT单总线控制 WS2812
GPIO 1光敏电阻 LDRIN (ADC)模拟量采集 0~4095
GPIO 2继电器1 (relay1)OUT预留扩展
GPIO 3继电器2 (relay2)OUT预留扩展
GPIO 4I²C SDA双向OLED 数据线
GPIO 5I²C SCLOUTOLED 时钟线
GPIO 6灯光继电器OUT控制照明设备
GPIO 7风扇继电器OUT控制散热风扇
GPIO 9板载 LEDOUT状态指示灯
GPIO 10DS18B20双向OneWire 数字温度
GPIO 20KEY1 总闸IN内部下拉, 轮询
GPIO 21KEY2 模式IN (INT)外部中断 RISING, 长短按

MQTT 通信协议

远程监控与控制的数据交互规范

📤

状态上报

chemctrl/{deviceId}/status

JSON: temperature, light, mode, key1_lock, relay3, relay4, temp_threshold, light_threshold

上报频率: 每5秒定期 + 状态变化即时

📥

命令订阅

chemctrl/{deviceId}/command

支持命令: set_relay, set_mode, get_status, set_threshold, reboot

⚙️

连接参数

🌐 外网

Broker: 47.98.170.180:8081

User: dzdx_emqx

🏠 内网

Broker: 192.168.199.219:1883

无需用户名密码（匿名）

KeepAlive: 15s | QoS: 0
ClientID: {deviceId}_{random}

传感器阈值设定

默认可动态调整的自动控制阈值（掉电保存于 Flash）

参数 默认值 触发动作 恢复条件
光照 ON 阈值 < 260 lux 开启灯光 光照 > 360 lux (OFF阈值)
光照 OFF 阈值 > 360 lux 关闭灯光 光照 < 260 lux (ON阈值)
温度 ON 阈值 > 30.5 °C 开启风扇 温度 < 29.5 °C (OFF阈值)
温度 OFF 阈值 < 29.5 °C 关闭风扇 温度 > 30.5 °C (ON阈值)

💡 阈值均支持通过串口命令 mqtt set threshold 或 MQTT set_threshold 命令动态修改
